## About UnstableSMP

UnstableSMP is a powerful Spigot/Paper plugin designed to enhance multiplayer survival servers with advanced features including:

- **Disguise System** - Change player skins and names
- **Resource Pack Management** - Automatic resource pack handling with a waiting room
- **Player Location Persistence** - Save and restore player locations across sessions
- **Adaptive Render Distance** - Optimize server performance automatically
- **Auto-Updating** - Automatic plugin updates with admin notifications
- **Combat Balancing** - Balanced weapon mechanics and enchantment management

## Getting Started

### For Players
- Join the server and experience the enhanced gameplay features
- Use `/disguise` to change your appearance (admin only)
- Enjoy optimized server performance with adaptive render distance

### For Server Admins
1. [Install the plugin](Installation)
2. Configure `config.yml` to your preferences
3. Set your resource pack URL in the config
4. Restart the server
5. Check `/unstable version` to verify installation

### For Developers
- View the source code on [GitHub](https://github.com/ResoluteStudios/UnstableSMP)
- Plugin built with **Paper API** and **Adventure API**
- Uses **SQLite** for player data persistence
